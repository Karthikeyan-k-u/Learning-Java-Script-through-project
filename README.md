## 📘 Overview

This repository provides a structured, project-based approach to learning core JavaScript concepts. Each task isolates a specific language feature—from variables to closures—and demonstrates its behavior through interactive browser-based examples. The emphasis is on **hands-on learning**: you see the code, you run the code, and you understand the output.

## 📋 Task Index & File Descriptions

This table outlines the purpose of each task folder and the specific files contained within.

| Task | Folder Name | Files | Description |
| :--- | :--- | :--- | :--- |
| **37** | `09-02-2026 TASK 37` | `js-intro.html`<br>`js-intro.js` | **Introduction to JavaScript**<br><br>• `js-intro.html`: Demonstrates two methods of including JavaScript: inline within a `<script>` tag and external file linking.<br>• `js-intro.js`: Contains a simple `alert()` function and `console.log()` statement to verify that external scripts execute correctly. |
| **38** | `13-02-2026 TASK 38` | `variables.html`<br>`variables.js` | **Variable Declaration**<br><br>• `variables.html`: Provides buttons that trigger the display of variable values on the webpage.<br>• `variables.js`: Defines variables using `var`, `let`, and `const` to illustrate differences in reassignment and block scope. |
| **39** | `16-02-2026 TASK 39` | `datatypes.html`<br>`datatypes.js` | **Data Types & Arithmetic Operators**<br><br>• `datatypes.html`: A simple interface for displaying operation results.<br>• `datatypes.js`: Works with string, number, and boolean types. Performs addition and multiplication operations to show how JavaScript handles type coercion. |
| **40** | `16-02-2026 TASK 40` | `conditionals.html`<br>`conditionals.js` | **Conditional Logic**<br><br>• `conditionals.html`: Accepts user input (a number) via a prompt or input field.<br>• `conditionals.js`: Implements `if...else if...else` to check if the number is positive, negative, or zero. Also includes a `switch` statement demonstration for day-of-week mapping. |
| **41** | `19-02-2026 TASK 41` | `loops.html`<br>`loops.js` | **Iteration & Loops**<br><br>• `loops.html`: Contains containers for dynamically generated list items.<br>• `loops.js`: Showcases the three primary loop structures: `for` (known iterations), `while` (condition-based), and `do...while` (guaranteed first execution). |
| **42** | `19-02-2026 TASK 42` | `functions.html`<br>`functions.js` | **Function Declarations**<br><br>• `functions.html`: Features input fields for two numbers and a calculate button.<br>• `functions.js`: Defines a traditional function `add()` that accepts parameters, performs a calculation, and returns the result to update the DOM. |
| **43** | `20-02-2026 TASK 43` | `arrow-functions.html`<br>`arrow-functions.js` | **ES6 Arrow Functions**<br><br>• `arrow-functions.html`: Accepts a number input to calculate its square.<br>• `arrow-functions.js`: Uses the concise arrow syntax `() => {}` to create a function that returns the square of a number. Highlights the implicit return feature. |
| **44** | `20-02-2026 TASK 44` | `scope-closures.html`<br>`scope-closures.js` | **Scope & Closures**<br><br>• `scope-closures.html`: Includes a counter display and an increment button.<br>• `scope-closures.js`: Differentiates between global, function, and block scope. Implements a **closure**—an inner function that retains access to an outer function's `count` variable even after the outer function has returned. |
